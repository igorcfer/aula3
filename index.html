Aula 3 - Git

Tópicos da Aula

  • O que é Git?
    • É um sistema de controle de versões gratuito e de código aberto.
  • O que é github?
    • um excelente serviço de hospedagem de repositórios de
      software com este sistema.
  • Acessando o terminal
  • Terminal no VsCode
  • Verificando a versão do git
    • git version
  • Configurando o Git pela primeira vez
    • git config --global user.name "Maria da Silva"
    • git config --global user.email maria@gmail.com
  • Verificando sua configuração
    • git config user.name
    • git config user.email
  • O que é um repositorio?
    • Um repositório é um espaço centralizado onde se armazena, organiza,
      mantém e divulga informação.
    • Será “a pasta” ou espaço na qual você guardará o seu projeto, para depois
      compartilhá-lo com a equipe por meio de um repositório na nuvem, por exemplo,
      no Github.
